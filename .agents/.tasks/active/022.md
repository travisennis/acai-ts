# Address 3 high-severity transitive npm advisories

## Problem

`npm audit --omit=dev` reports 3 HIGH-severity vulnerabilities in transitive dependencies:

- **Hono**: JWT middleware accepts any Authorization scheme (GHSA-f577-qrjj-4474); `app.mount()` strips prefix using undecoded path (GHSA-2gcr-mfcq-wcc3).
- **undici** (7.0.0-7.23.0): WebSocket 64-bit length overflow (GHSA-f269-vfmq-vjvj), HTTP request smuggling (GHSA-2mjp-6q6p-2qxm), CRLF injection via `upgrade` option (GHSA-4992-7rv2-5pvq), plus 3 more.

Both are pulled in transitively via `@ai-sdk/devtools` and the core AI SDK.

## Relevant files

- `package.json` — direct dependencies
- `package-lock.json` — transitives

4. Run `npm install` and verify `npm audit` is clean (or reduced).
