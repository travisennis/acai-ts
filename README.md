## Repo Cleanup

Sometimes it necessary to clean up your local repo. To do so, follow these steps:

```bash
# From the project root
npx npkill

# Then run the clean script
npm run clean```

Tool: [voidcosmos/npkill](https://github.com/voidcosmos/npkill)
